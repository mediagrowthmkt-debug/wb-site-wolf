# Wolf Carpenters Website

Website institucional da Wolf Carpenters - Carpintaria e Serviços de Construção.

## Estrutura do Projeto

- `index.html` - Página inicial
- `about.html` - Sobre nós
- `services.html` - Serviços
- `blog.html` - Blog
- `contact.html` - Contato
- `css/` - Arquivos de estilo
- `js/` - Scripts JavaScript
- `images/` - Imagens do site
- `videos/` - Vídeos e recursos multimídia

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Executar

Basta abrir o arquivo `index.html` em um navegador web moderno.
